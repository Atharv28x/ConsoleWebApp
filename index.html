Visual Studio Code

Microsoft Visual Studio is an integrated development environment from Microsoft.
It is used to develop computer programs,as well as websites, web apps, web services and mobile apps.

SetUp of Visual Sudio Code:

1.Download the visual studio Code installer for Windows.
2.Once it is downloaded,run the installer(VSCodeUserSetup-{version}.exe).This will only take a minute.
3.By Defaualt,vscode is installed

Steps to create and open a simple webpage using Visual Studio:

1.Create a folder on your computer for your project. name the folder portfolio.
2.Open the File menu and select Open Folder.
3.Open the File menu and select New File.
4.Right-click below the folder and select New File.
5.Now you have a blank text file named Index.

About Console Screen
